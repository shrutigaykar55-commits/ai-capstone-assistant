# 🤖 AI Capstone Assistant  
### Secure Desktop AI Assistant with Face Authentication and Smart GUI

A professional **end-to-end AI desktop assistant** built using **Python**, **OpenCV**, and **CustomTkinter**.

This project combines **computer vision**, **GUI development**, **rule-based AI interaction**, and **desktop automation** into a single secure application.

The system authenticates the user through **face detection**, displays a **secure splash screen**, and launches a **modern AI dashboard** where users can interact using text commands.

---

## 🚀 Key Features

- 🔐 Face authentication login using OpenCV
- 🎨 Modern dark-themed GUI dashboard
- 🖥 Professional sidebar-based layout
- ⚡ Animated splash / loading screen
- 🕒 Smart commands for date and time
- 🌐 Browser and YouTube automation
- 💬 Intelligent fallback chatbot responses
- 📊 Live status indicators and progress bars
- 🧹 Clear output and command history support
